========================================
NOTES
========================================

- Start with the QUICK SMOKE TEST first
- Watch the debug output: [RAW], [FAST], [NLU], [ENTITY]
- Some commands need files to exist (file search, content search)
- First screenshot creates the /captures folder
- Brightness may not work on external monitors
- System power commands will actually lock/shutdown!
- Type "exit" or "quit" to close the assistant
